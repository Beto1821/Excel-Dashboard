# Dashboard de Vendas - Randstad Análise de Dados

## Descrição do Projeto

Este projeto foi desenvolvido como parte do bootcamp **Randstad - Análise de Dados**. O desafio consistiu na criação de um dashboard de vendas em Excel para transformar dados brutos em insights visuais, auxiliando na análise de performance e na tomada de decisões estratégicas.

## Objetivo do Dashboard

O principal objetivo deste dashboard é proporcionar uma visualização clara e interativa dos dados de vendas. Com ele, é possível:

- Acompanhar os principais indicadores de desempenho (KPIs) de vendas.
- Analisar as vendas por diferentes categorias, como região, produto ou período.
- Identificar tendências, padrões e oportunidades de melhoria.
- Facilitar a comunicação dos resultados para as partes interessadas.

## Como Utilizar

1.  **Abra o arquivo:** Faça o download e abra o arquivo `../dio dash excel.xlsx` no Microsoft Excel.
2.  **Navegue pelas abas:** O arquivo pode conter diferentes planilhas para os dados brutos, tabelas dinâmicas e o dashboard principal.
3.  **Interaja com os filtros:** Utilize os filtros e segmentações de dados (slicers) disponíveis no dashboard para explorar as informações de acordo com suas necessidades.
